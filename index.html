1. Что такое HTML и его назначение

HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц. Он говорит браузеру, где заголовок, где текст, где картинка.

Пример простейшей страницы:

<!DOCTYPE html>
<html lang="ru">
<head>
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
</body>
</html>

2. Структура HTML-документа

Каждый HTML-файл должен начинаться с обязательных тегов. Это «скелет» страницы.

Основная структура:

<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8">
        <title>Заголовок вкладки</title>
    </head>
    <body>
        <!-- Здесь весь видимый контент -->
    </body>
</html>

3. Семантические заголовки

Заголовки создают структуру страницы. На всей странице должен быть только один <h1>.

Примеры заголовков:

<h1>Главный заголовок</h1>
<h2>Раздел</h2>
<h3>Подраздел</h3>

4. Текст, списки и изображения

Основные элементы для содержания:

  • <p> — обычный абзац
  • <ul> и <ol> — списки
  • <img> — изображение (обязательно указывай alt!)

Пример изображения:

<img src="https://picsum.photos/id/1015/600/300" 
     alt="Красивый пейзаж" 
     width="600">

5. Ссылки и навигация

Тег <a> создаёт ссылки.

Пример ссылки:

<a href="https://yandex.ru" target="_blank">Перейти на Яндекс</a>
Перейти на Яндекс (откроется в новой вкладке)

6. Интерактивные формы

Формы позволяют пользователям отправлять данные.

Пример формы:

Сообщение:

(Форма отправляется на тестовый сервис для проверки)

7. Полезные советы и лучшие практики

  • Всегда указывай alt у картинок
  • Используй семантические теги: <header>, <main>, <footer>
  • Проверяй код на validator.w3.org
  • Делай сайт адаптивным (добавляй meta viewport)
  • Не пропускай уровни заголовков

Помни: чистый и правильный HTML — основа любого хорошего сайта!
